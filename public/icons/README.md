# FormForge Icons

Place the following icon files in this directory:

- `icon-16.png` (16x16 pixels)
- `icon-48.png` (48x48 pixels)
- `icon-128.png` (128x128 pixels)

You can generate these using any image editor or use these placeholder commands:

```bash
# Using ImageMagick (if installed)
convert -size 16x16 xc:"#4f46e5" icon-16.png
convert -size 48x48 xc:"#4f46e5" icon-48.png
convert -size 128x128 xc:"#4f46e5" icon-128.png
```

Or use online tools to create simple icons with the FormForge branding.
